Memory allocation adalah proses pemberian ruang memori (RAM) untuk menyimpan data yang dibutuhkan selama program berjalan. 2 jenis dari memory allocation adalah static dan dynamis. static memory allocation terjadi saat awal compilation dan dynamic terjadi saat runtime. dalam memory allocation ada register variable dan external variable. void * adalah pointer generik yang bisa dikonversi ke tipe data lainnya, terutama digunakan saat alokasi memori secara dinamis. di memory allocation ada fungsi khusus seperti macro dan preprocessor untuk membantu dalam kompilasi bersyarat dan mendefinisikan nilai tetap dan command line argument input eksternal ke program dan pointer ke fungsi yang memungkinkan program fleksibel.
